## SAWS

Dockerized SAWS

[https://github.com/donnemartin/saws](https://github.com/donnemartin/saws)


## Add an alias to your .bashrc / .zshrc

```
alias saws="docker run -it --rm -v /Users/joshdvir/.aws:/root/.aws joshdvir/saws"
```
